# Blood Bank App

Blood Bank app built on **React Native** and **Firebase Realtime Database**.

## LANGUAGES USED:
1. **HTML**
2. **CSS**
3. **JAVASCRIPT**

## TECHNOLOGIES USED:
1. **REACT NATIVE** (FOR DEVELOPMENT PURPOSE)
2. **FIREBASE** (FOR DATABASE PURPOSE)
